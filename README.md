# 📝 예비 시험 문제 풀이 (Exam Quiz Generator)

`.txt` 또는 `.js` 문제 파일을 업로드하면 **자동으로 웹 시험으로 변환**해 주는 정적 웹앱입니다.
빌드 도구·서버 없이 `index.html`만 브라우저로 열면 바로 동작합니다.

객관식과 주관식을 모두 지원하며, 한 문제씩 풀고 마지막에 정답률을 집계합니다.

---

## ✨ 주요 기능

- **파일 업로드** — `.txt` / `.js` 파일을 드래그앤드롭하거나 선택해서 불러오기
- **자동 문제 변환** — 파일 형식을 인식해 객관식·주관식 문제로 변환
- **객관식** — 보기를 선택하면 즉시 정답/오답을 색으로 표시하고, 보기별 해설 제공
- **주관식** — 답안 작성 후 모범답안을 확인하고 `⭕ 맞음 / ❌ 틀림` 자가 채점
- **결과 집계** — 전체 정답 수와 정답률(%), 문제별 결과 목록 표시
- **다시 보기** — 결과 화면에서 특정 문제로 바로 이동해 복습

---

## 🚀 사용 방법

1. 이 저장소를 클론하거나 다운로드합니다.
2. `index.html` 을 브라우저로 엽니다. (별도 설치·빌드 불필요)
3. 시작 화면에서 문제 파일(`문제.txt` 또는 `sample-questions.js`)을 업로드합니다.
4. **시험 시작** 을 눌러 문제를 풉니다.

> 정적 파일만으로 동작하므로 GitHub Pages 등에 그대로 배포할 수 있습니다.

## 📂 폴더 구조

```
시험 문제 생성기/
├─ index.html            단일 페이지 (시작 / 풀이 / 결과 화면 + 출제규칙 모달)
├─ RULE.md               문제 파일 작성 규칙 문서
├─ README.md             프로젝트 설명 (이 파일)
├─ 문제.txt              샘플 문제 (소프트웨어 공학 60문항)
├─ sample-questions.js   JS 형식 샘플 문제
└─ src/
   ├─ parser.js          .txt / .js 파서 (DOM 비의존)
   ├─ quiz.js            풀이 엔진 (파일 로딩 · 채점 · 결과 집계)
   └─ style.css          스타일
```

## 📋 문제 파일 형식

문제 파일은 **TXT 형식**과 **JS 형식** 두 가지를 지원합니다. 자세한 작성 규칙은 [`RULE.md`](RULE.md) 또는 앱 내 **📋 출제 형식 보기** 버튼에서 확인할 수 있습니다.

### TXT 형식 (인라인) 예시

```
Q1. UML이 제시하는 다이어그램의 총개수는?
① 5개
② 9개
③ 12개
④ 14개
정답: ②
해설: UML은 9개의 다이어그램을 제시합니다.

[주관식] 소프트웨어 공학의 정의를 서술하시오.
모범답안: 소프트웨어를 체계적·정량적으로 개발·운용·유지보수하는 학문.
```

### JS 형식 예시

```js
window.QUIZ_QUESTIONS = [
  {
    type: "multiple",
    question: "질문 내용",
    choices: ["보기1", "보기2", "보기3", "보기4"],
    answer: 2,                  // 정답 보기 번호 (1부터 시작: 1=①)
    explanation: "전체 해설",
    choiceExplanations: ["①틀린 이유", "②정답 이유", "③틀린 이유", "④틀린 이유"]
  },
  {
    type: "short",
    question: "질문 내용",
    modelAnswer: "모범 답안",
    explanation: "채점 참고용 키워드"
  }
];
```

| 형식 | 객관식 | 주관식 | 보기별 해설 | 작성 난이도 |
|------|:------:|:------:|:-----------:|:-----------:|
| TXT (인라인) | ✅ | ✅ | ✅ | 쉬움 |
| TXT (레거시 분리형) | ✅ | ❌ | ❌ | 보통 |
| JS | ✅ | ✅ | ✅ | 보통 (권장) |

## 🛠 기술 스택

- HTML5 / CSS3 / Vanilla JavaScript (프레임워크·빌드 도구 없음)
- 브라우저 `FileReader` API 로 파일 로딩 (UTF-8)
- 파서는 DOM 비의존으로 작성되어 Node 환경에서도 재사용 가능

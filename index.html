📋 시험 문제 출제 규칙

이 규칙대로 파일을 만들면 사이트에 넣었을 때 그대로 시험으로 변환됩니다. 형식은 두 가지(.txt / .js), 문제 유형은 객관식과 주관식 두 가지입니다.

1. TXT 형식

1-1. 권장(인라인) 형식 — 객관식

각 문제는 빈 줄로 구분하고, 정답·해설을 문제 바로 아래에 적습니다.

Q1. 질문 내용을 여기에 적습니다.
① 첫 번째 보기
② 두 번째 보기
③ 세 번째 보기
④ 네 번째 보기
정답: ③
해설: 왜 ③번이 정답인지 설명합니다.
보기해설:
① 이 보기가 틀린 이유
② 이 보기가 틀린 이유
③ 이 보기가 정답인 이유
④ 이 보기가 틀린 이유
  • 문제 번호: Q1. 1. 1) 모두 인식
  • 보기 기호: ①②③④… 또는 1) 2) 3)
  • 정답: 에는 기호(③) 또는 숫자(3)를 적습니다.
  • 해설: 과 보기해설: 은 선택 사항입니다. 보기해설이 있으면 각 보기 아래에 개별 설명이 표시됩니다.

1-1. 권장(인라인) 형식 — 주관식

[주관식] 질문 내용을 여기에 적습니다.
모범답안: 모범 답안 내용. 여러 줄도 가능합니다.
해설: 채점 시 참고할 핵심 키워드 (선택).
  • 맨 앞에 [주관식] 을 붙이거나, 모범답안: 줄이 있으면 자동으로 주관식으로 인식됩니다.
  • 보기(①②③④)가 없는 문제도 자동으로 주관식 처리됩니다.

1-2. 분리(레거시) 형식 — 기존 문제.txt 호환

문제를 모두 먼저 쓰고, 맨 아래에 정답 및 해설 구분선을 두는 형식도 지원합니다.

Q1. 질문...
① 보기  ② 보기  ③ 보기  ④ 보기

Q2. 질문...

=========================================================
정답 및 해설
=========================================================

Q1. 정답: ③
해설: 설명...
암기 팁: ...

Q2. 정답: ①
해설: 설명...
  • 정답 및 해설 문구가 든 줄을 기준으로 문제부/정답부를 나눕니다.
  • 정답부의 Q1., Q2. 번호 순서대로 위 문제와 1:1로 연결됩니다.
  • 암기 팁: 은 해설에 함께 표시됩니다.

⚠️ 레거시 형식은 객관식만 표현할 수 있습니다. 주관식·보기별 해설이 필요하면 인라인 형식이나 JS 형식을 사용하세요.

2. JS 형식 (권장)

전역 배열 QUIZ_QUESTIONS 를 정의합니다. (가장 안전: window.QUIZ_QUESTIONS)

window.QUIZ_QUESTIONS = [
  // ── 객관식 ──
  {
    type: "multiple",
    question: "질문 내용",
    choices: ["보기1", "보기2", "보기3", "보기4"],
    answer: 3,                 // 정답 보기 번호 (1부터: 1=①, 2=②, 3=③)
    explanation: "전체 해설",    // 선택
    choiceExplanations: [       // 선택. 각 보기 아래에 표시
      "보기1이 틀린 이유",
      "보기2가 틀린 이유",
      "보기3이 정답인 이유",
      "보기4가 틀린 이유"
    ]
  },
  // ── 주관식 ──
  {
    type: "short",
    question: "질문 내용",
    modelAnswer: "모범 답안 (여러 줄 가능)",
    explanation: "채점 참고용 핵심 키워드 (선택)"
  }
];
필드유형필수설명
type"multiple" / "short"권장생략 시 choices가 있으면 객관식, 없으면 주관식
question문자열✅문제 본문
choices문자열 배열객관식 ✅보기 목록
answer숫자객관식 ✅정답 보기 번호. 1부터 시작 (1=①)
explanation문자열선택전체 해설
choiceExplanations문자열 배열선택보기 순서대로 1:1, 각 보기 아래 표시
modelAnswer문자열주관식 ✅모범 답안

3. 동작 방식 요약

  • 한 페이지에 한 문제씩 표시됩니다.
  • 객관식: 보기를 선택하면 즉시 정답/오답이 색으로 표시되고, 각 보기 아래에 (있다면) 해설이 나옵니다.
  • 주관식: 답을 제출하면 모범 답안이 나오고, ⭕ 맞음 / ❌ 틀림 으로 직접 채점합니다.
  • 모든 문제를 풀면 정답 수 / 전체 문항 수 와 정답률(%) 이 표시됩니다.
